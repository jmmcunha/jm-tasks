Integração com Planner

Ao tocar em Planner em qualquer tarefa, o app abre um formulário Microsoft Forms já pré-preenchido com título, OE/bucket, prioridade, datas e anotações. Você só toca em Enviar, e o Power Automate cria a tarefa no plano DE - Geral, no bucket correto, sem caracteres HTML.

  • Bucket por OE: definido pelo nome da opção no Forms (já fixo).
  • Prioridade: Urgente / Importante / Média / Baixa (mapeada automaticamente da prioridade do app).
  • Datas: Data de início (default: hoje) e Prazo (data de conclusão).
  • ID do app: enviado junto para futura sincronização de status.

Proteger este aplicativo com senha

Antes de cadastrar tarefas, defina uma senha para criptografar os dados deste navegador.

Aviso de privacidade
  • Os dados ficam apenas neste navegador, em armazenamento local. Não há servidor, login externo nem nuvem.
  • Tudo é cifrado com a sua senha (AES-GCM 256, derivada via PBKDF2 com 200 mil iterações).
  • Não há recuperação de senha. Se você esquecer, não tem como recuperar — só apagar tudo e começar de novo.
  • Faça backup periódico em Exportar JSON para não perder os dados.
  • Cada navegador tem sua base própria — quem abrir o link em outro dispositivo começa do zero, com sua própria senha.
