Migração: dados pessoais → equipe

Esta página faz uma operação única: copia seus dados antigos (em users/{seu-uid}/data/) para o novo caminho compartilhado (shared/team/data/) que toda a equipe vai usar.

Passo 1. Faça login com a conta cuja agenda você quer usar como base da equipe.

Passo 2. Após logar, clique para copiar os dados.
